# Service Booking App

Aplicativo mobile de agendamento de serviços, permitindo que profissionais gerenciem sua agenda e que clientes realizem agendamentos de forma rápida e prática.

## 📱 Funcionalidades

* Autenticação de usuários (login e cadastro)
* Agendamento de serviços
* Visualização de horários disponíveis
* Cancelamento e remarcação de agendamentos
* Gerenciamento da agenda do profissional
* Lembretes automáticos (planejado)

## 🛠️ Tecnologias

* Android Studio
* Kotlin
* Firebase (planejado para backend e banco de dados)

## 🎯 Objetivo

O objetivo deste projeto é fornecer uma solução simples e eficiente para o gerenciamento de agendamentos de serviços, melhorando a organização dos profissionais e oferecendo mais praticidade para os clientes.

## 🧑‍💼 Público-Alvo

* Barbeiros
* Cabeleireiros
* Manicures
* Esteticistas
* Personal trainers
* Prestadores de serviços em geral
